# 台幣換他國貨幣 - Code Review 示範程式

## 功能

這是一個簡單的網頁應用程式，可以將台幣轉換為各國貨幣（美元、日圓、歐元、英鎊、人民幣）。

**使用方式：**
1. 輸入要轉換的台幣金額
2. 選擇要轉換成的目標貨幣
3. 點擊「轉換」按鈕
4. 顯示轉換後的外幣金額

---

## 程式目的

⚠️ **這是一個示範程式，用於展示 GitHub Copilot 的 Code Review 功能。**

### 故意埋入的問題類型

為了完整示範 Copilot 的 Code Review 能力，程式碼中故意包含了以下常見的程式問題：

### 1. **安全漏洞**
- ❌ API Key 直接暴露在前端代碼中
- ❌ XSS 漏洞：使用 `innerHTML` 直接渲染使用者輸入，未進行 escape
- ❌ 缺少輸入驗證

### 2. **邏輯 Bug**
- ❌ 匯率計算中有奇怪的 `+100` 邏輯
- ❌ 沒有進行結果四捨五入
- ❌ API 調用時序不對（計算完後才調用 API）

### 3. **代碼品質差**
- ❌ 變數命名不規範：`x`, `y`, `z`, `temp_data`
- ❌ 存在未使用的廢棄代碼：`oldConvertMethod()`
- ❌ 缺少錯誤處理
- ❌ 全域變數亂用

### 4. **效能問題**
- ❌ 沒有 debounce 機制（頻繁觸發 API 調用）
- ❌ 硬編碼的匯率值和 API 調用混用

## 注意事項

⚠️ **這個程式不適合用於生產環境！**  
它只是用於教學和示範目的。正式應用應該：
- 移除硬編碼的 API Key
- 修復所有 XSS 漏洞
- 添加完整的輸入驗證
- 改進變數命名和代碼結構
- 實現適當的錯誤處理
